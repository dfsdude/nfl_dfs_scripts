# Services Directory

## Purpose
Shared service layer for cross-cutting concerns and reusable algorithms.

## Files
- `simulations.py` - Simulation algorithms and statistical methods
- `__init__.py` - Package initialization

## Responsibilities
- Monte Carlo simulation logic
- Probability calculations
- Shared statistical functions
- Algorithms used by multiple tools

## Usage
Services are imported by modules when they need simulation or statistical computation capabilities.
